HTML нь вэб хуудасны стандарт тэмдэглэгээний хэл юм. HTML-ийн тусламжтайгаар та өөрийн вэбсайтыг үүсгэж болно. HTML сурахад хялбар - Танд таалагдах болно!

Жишээ код:
<!DOCTYPE html>
<html>
<head>
<title>Вэб хуудасны гарчиг</title>
</head>
<body>
<h1>Энэ бол гарчиг</h1>
<p>Энэ бол догол мөр.</p>
</body>
</html>

HTML танилцуулга, HTML бол вэб хуудас үүсгэх стандарт тэмдэглэгээний хэл юм.
HTML гэж юу вэ?
• HTML нь Hyper Text Markup Language гэсэн үгийн товчлол юм.
• HTML бол вэб хуудас үүсгэх стандарт тэмдэглэгээний хэл юм.
• HTML нь вэб хуудасны бүтцийг тодорхойлдог.
• HTML нь хэд хэдэн элементүүдээс бүрдэнэ.
• HTML элементүүд нь вэб хөтөчид агуулгыг хэрхэн харуулахыг хэлж өгдөг.
• HTML элементүүд нь "Энэ бол гарчиг", "Энэ бол догол мөр.", "Энэ бол холбоос" гэх мэт агуулгын хэсгүүдийг шошгодог.

Энгийн HTML баримт бичиг. Жишээ код:
<!DOCTYPE html>
<html>
<head>
<title>Вэб хуудасны гарчиг</title>                              
</head>
<body>
<h1>Миний анхны гарчиг heading</h1>
<p>Миний эхний догол мөр paragraph.</p>
</body>
</html>

Жишээ код тайлбарлав.
• Тунхаглал <!DOCTYPE html> нь энэ баримт бичиг нь HTML5 баримт бичиг гэдгийг тодорхойлдог.
• <html> элемент нь HTML хуудасны үндсэн элемент юм.
• <head> элемент нь HTML хуудасны тухай мета мэдээллийг агуулдаг.
• <title> элемент нь HTML хуудасны гарчгийг тодорхойлдог. Энэ нь хөтчийн гарчгийн мөрөнд эсвэл хуудасны таб дээр харагдах болно.
• <body> элемент нь баримт бичгийн биеийг тодорхойлдог бөгөөд гарчиг, догол мөр, зураг, гипер холбоос, хүснэгт, жагсаалт гэх мэт харагдахуйц бүх агуулгыг хадгалах сав юм.
• <h1> элемент нь том гарчгийг тодорхойлдог.
• <p> элемент нь догол мөрийг тодорхойлдог.

HTML элемент гэж юу вэ?
HTML элемент нь эхлэх шошго, зарим агуулга, төгсгөлийн таагаар тодорхойлогддог:

<tagname>Агуулга энд байна...</tagname>

HTML элемент нь эхлэл хаягаас эхлээд төгсгөлийн шошго хүртэлх бүх зүйл юм.

<h1>Миний анхны гарчиг</h1>
<p>Миний эхний догол мөр.</p>

<h1>My First Heading</h1>

[Start tag] [Element content] [End tag]

Start tag буюу эхлэлийн тааг бол <h1> юм.
Харин End tag болох төгсгөлийн тааг нь </h1>
Эхлэлийн тааг болон төгсгөлийн таагны голд нь элементийн агуулга орох болно.

<эхлэх-тааг>элементийн агуулга...</төгсөх-тааг>

<br> бол эхлэлийн тааг элемент агуулга болон төгсгөлийн тааг байхгүй болно.

Тайлбар: Зарим HTML элементүүдэд ямар ч агуулга байхгүй (<br> элемент гэх мэт). Эдгээр элементүүдийг хоосон элементүүд гэж нэрлэдэг. Хоосон элементүүдэд төгсгөлийн тэмдэг байхгүй!

Вэб хөтчүүд, Вэб хөтчийн зорилго
(Google Chrome, Microsoft Edge, Mozilla Firefox, Apple Safari) нь HTML баримтуудыг уншиж, тэдгээрийг зөв харуулах явдал юм. Хөтөч нь HTML хаягуудыг харуулахгүй, харин баримтыг хэрхэн харуулахыг тодорхойлоход ашигладаг:

HTML хуудасны бүтэц, HTML хуудасны бүтцийн дүрслэлийг доор харуулав:

<html>
  <head>
   <title>Webpage Title</title>
  </head>
  <body>
<h1>This is a heading</h1>
<p>This is a paragraph.</p>
<p>This is another paragraph.
</p>
  </body>
</html>

Тайлбар: <body> хэсгийн доторх контент хөтөч дээр харагдах болно. <title> элементийн агуулгыг хөтчийн гарчгийн мөрөнд эсвэл хуудасны таб дээр харуулах болно.
